# Tokyo Night Prompt Spec

Prompt order:

1. OS
2. Current directory
3. Git branch and status
4. Command duration
5. Time
6. Prompt character

Color anchors:

* OS: `#7aa2f7`
* Directory: `#7dcfff`
* Git branch: `#bb9af7`
* Git status and duration: `#e0af68`
* Time: `#565f89`
* Success: `#9ece6a`
* Error: `#f7768e`
